#  Pet Store CLI Application

This is a basic command-line Python project created for the course **COMP 2120 – Logic Programming**.

#  Description
This CLI app simulates a pet store. Users can:

- View current animal inventory
- Purchase one animal per species
- View past purchases
- Exit the application

Each purchase includes the user’s full name and the purchase timestamp using Python's `datetime`.

#  Concepts Applied

- Conditionals
- Loops
- Functions
- Dictionaries
- User input
- Basic data handling
